{{ _('Rules & Prizes') }}

{% trans %}Understand how Arena Eureka works, how scoring is calculated and what the prize criteria are.{% endtrans %}

{{ _('How it works') }}

  • {% trans %}. Each participant can make predictions on the matches available in each phase.{% endtrans %}
  • {% trans %}. Predictions can be changed until the official start of the match.{% endtrans %}
  • {% trans %}. After the game begins, the prediction is locked.{% endtrans %}
  • {% trans %}. The score is automatically calculated based on the actual result.{% endtrans %}

{{ _('Scoring') }}

  • {{ _('10 points:') }} {% trans %}correctly predicted the exact result - number of goals for both teams and the winner.{% endtrans %}
  • {{ _('7 points:') }} {% trans %}predicted the winner + the number of goals for one of the teams.{% endtrans %}
  • {{ _('5 points:') }} {% trans %}predicted only the winner.{% endtrans %}
  • {{ _('2 points:') }} {% trans %}predicted only the number of goals for one of the teams.{% endtrans %}
  • {{ _('0 points:') }} {% trans %}did not get any criteria correct.{% endtrans %}

{{ _('General rules') }}

  • {% trans %}. The ranking is updated based on results that have already been confirmed.{% endtrans %}
  • {% trans %}. The system may display an overall ranking as well as rankings by unit/audience.{% endtrans %}
  • {% trans %}. In case of a tie, the system uses internal criteria already defined in the ranking.{% endtrans %}
  • {% trans %}. Messages in Arena Talks must respect the community environment.{% endtrans %}

{{ _('Prizes') }}

  • {% trans %}. Prizes may vary depending on the campaign and the Eureka unit.{% endtrans %}
  • {% trans %}. The main prize will be based on the overall ranking of the competition.{% endtrans %}
  • {% trans %}. There may be highlights by unit or specific category.{% endtrans %}
  • {% trans %}. Final prize details will be communicated by each Eureka unit.{% endtrans %}

{{ _('The prediction is locked and can no longer be modified.') }}

{{ _('Can I follow my ranking position?') }}

{% trans %}Yes. The internal area shows your profile, position and score.{% endtrans %}
